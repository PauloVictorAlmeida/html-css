Outras formaçoes de escrtia

Texto marcado, utilizamos para deixar como se fosse um marcador

Texto grande, a formataçao ficar maior, podemos utilizar a tag, porem o recomendado é usar o css

Texto pequeno, a palavra fica menor, nesse caso so a tag ja resolve, nao precisa de css

Texto deletado, neste aqui teremos palavras que ficam cortadas ao meio para falar que nao deveriam esta la

Texto inserido/sublinhado, fica uma linha embaixo da palavra para fa um leve destaque, tem outra forma porem essa forma nao é sematica, entao melhor usar a primeira forma

Texto sobrescrito, fica a cima das palvras, é bom para uma possivel equaçao, ex: x20+3

Texto subcrito, fica embaixo das palavras, é bom para equaçoes quimicas, ex: H2O
